# UI Prompt Collection for Sales Admin Automation Toolkit

This pack contains prompt templates to guide coding agents, Figma AI / Figma Make, and Figma MCP workflows for designing and implementing the UI of the **Sales Admin Automation Toolkit**.

Project goal:

> Build a professional internal operations dashboard for sales admin / operations workflows: order validation, inventory allocation, payment aging, and Excel report export.

Recommended usage order:

1. Use `01_figma_ai_generation_prompts.md` to generate or refine UI screens in Figma Make / Figma AI.
2. Use `02_reference_to_design_prompts.md` when using screenshots or UI references.
3. Use `03_figma_mcp_agent_prompts.md` when coding agents can inspect Figma through MCP.
4. Use `04_nextjs_ui_implementation_prompts.md` when translating approved UI into Next.js / Tailwind / shadcn code.
5. Use `05_ui_review_and_acceptance_prompts.md` to critique and finalize the UI before implementation.

Important principle:

Do not let the UI design expand the project into a full ERP. Keep the project as a focused, Excel-first sales admin automation toolkit.
